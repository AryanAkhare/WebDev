WebDev — Learning Web Development

This workspace collects small web development projects and experiments organized by plain HTML/CSS/JavaScript examples and React apps.

**Repository Structure**
- core-web: Static web projects and small demos organized into subfolders:
	- `CSS/` — component and layout experiments (e.g., modern buttons, galleries, portfolio pages).
	- `HTML/` — HTML practice pages and small examples.
	- `Javascript/` — vanilla JS lessons and DOM exercises.
	- `SmallProjects/` — small apps like Calculator, ToDo, TicTacToe, Weather, PasswordGenerator.
- react-apps: React projects grouped by difficulty and topic:
	- `react-basics/` — starter apps and simple components.
	- `react-intermediate/` — form handling, routing, and intermediate patterns.
	- `react-advanced/` — context, state management (Redux), and more advanced examples.

**How to use**
- Open any project folder in your editor or run the demo by opening the HTML file in a browser.
- For React apps: change into the app folder, install dependencies with `npm install`, then run `npm run dev` or `npm start` depending on the project setup.
